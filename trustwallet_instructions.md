
# TrustWallet Logo Submission Guide

To add Kosovo Coin logo to TrustWallet:

1. Image requirements:
   - PNG format
   - 512x512 resolution
   - Transparent background
   - File name must match contract address and be lowercase:
     `0x094a23c55062dce69d6efba1ab9da271b9e5c9f3.png`

2. Place the file in:
   `assets/blockchains/smartchain/assets/0x094a23c55062dce69d6efba1ab9da271b9e5c9f3/logo.png`

3. Submit a pull request to:
   https://github.com/trustwallet/assets

4. Required additional files:
   - info.json with metadata
   - logo.png

This folder contains instructions only. Add your real logo file before submission.
